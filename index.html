Add a Second Agent

You can run multiple isolated agents — each with their own personality, workspace, and channel. Useful for separating personal vs. practice, or adding a Discord-specific agent.

💼 Practice Agent (example)
Dedicated to Momentum Health — monitors reviews, drafts social posts, answers patient FAQs via Discord
openclaw agents add practice

Then bind it to Discord and configure a separate workspace with practice-specific context.

📈 Finance Agent (example)
Runs daily market analysis, alerts on watchlist moves, summarizes earnings calls automatically
openclaw agents add finance
How to set up a new agent:
1. SSH into your server
2. Run openclaw agents add <name>
3. Follow the wizard to set workspace, channel, and persona
4. Message Jeffrey on Telegram to configure it further
